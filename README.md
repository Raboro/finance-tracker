## Using

Go the the [website](https://expo.dev/@raboro/financetracker) and scan the QR Code.

### Locally
1. Clone the repo
2. Install all dependencies: ``npm install``
3. Run ``npm start`` 
4. Scan QR Code 

## Update
Run ``eas update`` to upload the newest changes to expo.
